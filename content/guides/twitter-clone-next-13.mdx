## Error: LAMDA_RUNTIME Failed to post handler success response. Http response code: 413.

You need to limit the response. 
thanks to rubensoon @ https://discord.com/channels/1079557715497595013/1083833122216497192/1105237969834623086

add this in the api route.ts
TODO: Check which route.ts to add to, currently add to everyone. 

Location: pages/api/../route.ts files.

```javascript
export const config = {
  api: {
    responseLimit: false,
  },
}
```
